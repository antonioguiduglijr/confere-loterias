Código criado inspirado no repo: https://github.com/leandrotoledo/mega
====

Confere-Loterias
====
Script python para conferir resultado de concursos da loteira com apostas realizadas


Instruções
==========

1) No mesmo local do script criar arquivo mega.txt com as suas dezenas apostadas. 

Por exemplo:

04-07-12-25-37-43

02-13-20-22-31-42

02-09-12-16-28-33-43-46-51-52

03-06-12-29-31-38-50-51-57

02-05-07-11-17-34-42-54


2) Executar o script com o código do concurso:
python mega.py 2962


4) Se você ganhar, faça um donate pra mim! rsrs!
